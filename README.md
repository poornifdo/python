# Python Exercise

This repository contains small Python exercises for practice and learning.

## Setup

- Install Python 3.8+.
- (Optional) create and activate a virtual environment:
	- python -m venv venv
	- venv\Scripts\activate  # Windows

## Exercises

1. Basic syntax and data types
2. Functions and modules
3. File I/O
4. Simple algorithms and data structures

## Running

Run Python files from the command line, for example:

```
python exercise1.py
```
